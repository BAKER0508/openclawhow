# OpenClaw How - Project Rules

## Overview
This is a Next.js 14 site showcasing OpenClaw real-world cases and scene solutions. It uses the App Router, Tailwind CSS, and MDX content files.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- gray-matter (frontmatter parsing)
- MDX content files in src/content/

## Directory Structure
- `src/app/` - Pages and layouts (App Router)
- `src/components/` - Reusable UI components
- `src/lib/` - Utilities, types, data fetching
- `src/content/cases/` - Case study MDX files
- `src/content/scenes/` - Scene solution MDX files

## Design Tokens
- Primary (dark blue): #1a1a2e
- Accent (orange): #ff6b35
- Teal: #4ecdc4
- Surface (background): #f8f9fa

## Conventions
- Mobile-first responsive design
- Keep components simple, no over-engineering
- English UI, Chinese titles stored in titleZh field for future i18n
- All content lives in MDX files, no database needed
